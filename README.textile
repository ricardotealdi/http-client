h1. http-client

p. A simple http client for Java

h1. Usage

h2. Creating a Request using the builder

bc.. Request request =
    new RequestBuilder()
        .forThis("your-uri")
        .withHeaderProperty("Accept", "application/xml")
        .withHeaderProperty("Content-Type", "application/xml")
        .withBody("body-to-be-sent")
        .instance();

h2. Using the http client to send your request


h3. A GET request

bc.. Response response = 
    new HttpClient()
        .doGet(
            new RequestBuilder()
                .forThis("your-uri")
                .instance());

h3. A POST request

bc.. Response response =
    new HttpClient()
        .doPost(
            new RequestBuilder()
                .forThis("your-uri")
                .withHeaderProperty("Accept", "application/xml")
                .withHeaderProperty("Content-Type", "application/xml")
                .withBody("body")
                .instance());

h3. A PUT request

bc.. Response response =
    new HttpClient()

                .withHeaderProperty("Accept", "application/xml")
                .withHeaderProperty("Content-Type", "application/xml")
                .withBody("body")
                .instance());
